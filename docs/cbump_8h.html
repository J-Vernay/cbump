Makes box move along the given move vector, taking into account obstacles.

Parameters
boxThe box to move. Its position will be updated.
moveThe vector representing the movement, relative to current box position.
nb_obstaclesNumber of obstacles present in obstacles array.
obstaclesArray of pointers to potential obstacles which box may collide too. NULL-pointers are not allowed. It will be reordered in imprevisible ways (for internal use reasons): so you must not rely on order kept after calling cbump_move().
resolverThe resolver to use. Can be NULL, in which case cbump_slide is used.
userdataUser data provided to resolver. Not used internally.

cbump_vec2 cbump_time_of_impact ( cbump_box const *  box,
cbump_box const *  other,
cbump_vec2  move 
)

Returns the time of impact on both axis clamped between 0 and 1.

For each axis, returns the factor by which multiplying 'move' component will make 'box' and 'other' touch each other on this axis.

Parameters
boxThe box moving along move .
otherThe static box which may be collapsed with;
moveThe direction of movement. Time of impact is relative to 'move'.
Return values
Componentis one if move can be done without overlapping, or if already overlapping.
Componentbetween 0 and 1 giving the time of impact.
